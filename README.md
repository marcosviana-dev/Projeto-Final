🎯 Jogo de Precisão — Acerte o Alvo

Jogo desenvolvido em grupo durante a graduação em Análise e Desenvolvimento de Sistemas (2021).

Sobre o projeto

O objetivo é clicar no alvo em movimento 5 vezes no menor tempo possível. Um cronômetro registra o tempo da partida e uma mensagem de vitória é exibida ao final.

Tecnologias utilizadas


HTML5
CSS3
JavaScript (Canvas API)


Como rodar

Não precisa instalar nada. Basta baixar os arquivos e abrir o index.html no navegador.

Funcionalidades


Alvo gerado em posição aleatória a cada intervalo
Cronômetro em tempo real (HH:MM:SS)
Detecção de clique com cálculo de distância
Tela de vitória ao acertar 5 vezes
Botão de reiniciar


Aprendizados

Primeiro contato prático com lógica de programação aplicada no front-end, manipulação do Canvas API e eventos do DOM com JavaScript puro.
